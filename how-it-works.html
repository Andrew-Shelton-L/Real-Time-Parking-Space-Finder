Detection/Monitoring

The Monitoring process is running on a RPI kit which uses the concept of Motion Detection with the help of OpenCV where it regularly monitors the Marked Parking spaces.

  • Marking coordinates of each parking spaces

  • Finding the Region of Intrest(ROI)

  • Applying Laplacian on the ROI

  • Finding the mean value from the resulted image

  • Deciding the status as free or not

These status values are then sent to the Data Store (DataBase)

Data Store

The Data can be recieved with the help of an API which continuously updates the status for the respective parking space to the database in a remote host

Viewing

Finally with the help of the data from the database the user can view the status of the respective parking lot's Avilable spaces in a Web Browser or an Android app that you are using right now.
